# Three-Substrate Parity

This demonstrator records the parity contract across the reference, WGPU, and
SPIR-V substrates. The live GPU checks run in `.github/workflows/gpu-parity.yml`;
the SPIR-V parity smoke lives in `vyre-driver-spirv/tests/spirv_parity.rs`.

Published parity reports live under `docs/parity/`.
